# ProxyLens AI

Day 19 demo for `broadridge.asion.ai`.

ProxyLens AI is a shareholder voting intelligence agent concept for governance, issuer, stewardship, and shareholder communication teams. It uses synthetic proxy-season data to identify proposal type, surface governance red flags, map likely investor concerns, simulate vote support, draft neutral shareholder communications, and generate an evidence-backed Proxy Readiness Brief.

This is a concept demo. It is not affiliated with Broadridge and does not provide proxy voting advice.

## Workflows

- Proposal classification
- Governance red flag detection
- Vote risk scoring
- Investor concern mapping
- Communication studio
- Vote outcome simulation
- Proxy Readiness Brief export
